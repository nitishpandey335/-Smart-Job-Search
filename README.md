# Smart Job Search 🔍

A powerful AI-powered job search application that aggregates listings from multiple platforms with intelligent filtering and market insights.

![App Screenshot]()

## Features ✨

- **Multi-platform Search**: Unified search across LinkedIn, Indeed, and other portals
- **AI-Powered Suggestions**: Smart auto-complete for jobs and locations
- **Advanced Filters**:
  - Experience level (Fresher to 10+ years)
  - Salary range (0-3 LPA to 15+ LPA)
  - Job type (Full-time, Remote, etc.)
- **Real-time Market Insights**:
  - Trending skills with growth metrics
  - Salary benchmarks by role
  - Hot job locations
- **Company Explorer**: Curated list of featured employers
